## 推荐发布顺序

1. 首图：`00-cover-card.png`，负责抓眼球和说明定位。
2. 第二张：`01-dashboard.png`，证明项目能跑起来、不是纯文档；已替换为脱敏示例书籍。
3. 第三张：`03-book-workspace.png`，展示最核心的小说创作工作区；不展示你的当前书籍。
4. 第四张：`02-ai-chat.png`，展示对话式创作能力。
5. 第五张：`05-model-services.png`，展示模型服务商配置，增强“可落地”信任感。
6. 第六张：`06-codingplan-config.png`，突出中转站 / CodingPlan / Base URL 配置。
7. 第七张：`04-create-book.png`，展示从 0 创建项目的入口。
8. 第八张：`07-mobile-dashboard.png`，如果闲鱼图片格子足够，可作为补充；否则可不发。

## 图片文字建议

- 首图大字：AI 小说创作 Web 工作台
- 副标题：书籍管理 / 对话写作 / 章节审稿 / 模型配置
- 角标：源码交付｜可本地运行｜支持二开
- 避免：保证收益、爆款神器、全自动赚钱、独家源码等表述。
